# QL_CuaHangBanHoaQua
Một cửa hàng kinh doanh hoa quả tươi muốn xây dựng website đặt hàng trực tuyến để khách hàng có thể xem sản phẩm và đặt mua dễ dàng.
Hệ thống cho phép:
•	Khách hàng xem danh sách hoa quả
•	Đặt hàng online
•	Thanh toán
•	Theo dõi đơn hàng
Cửa hàng sử dụng hệ thống để:
•	Quản lý sản phẩm
•	Quản lý đơn nhập hàng
•	Quản lý đơn bán hàng
•	Quản lý nhân viên và tài khoản











Quy trình hoạt động của hệ thống
Bước 1:
Khách hàng truy cập website → xem danh sách hoa quả.
Bước 2:
Khách chọn sản phẩm → thêm vào giỏ hàng.
Bước 3:
Khách xác nhận đặt hàng → hệ thống tạo đơn hàng.
Bước 4:
Nhân viên xác nhận đơn → chuẩn bị hàng.
Bước 5:
Đơn hàng được giao cho khách.












Các chức năng chính của website
Khách hàng:
•	Đăng ký
•	Đăng nhập
•	Xem sản phẩm
•	Tìm kiếm hoa quả
•	Thêm vào giỏ hàng
•	Đặt hàng
•	Xem lịch sử mua
•	Theo dõi đơn hàng

Quản trị viên:
•	Quản lý sản phẩm
•	Quản lý đơn hàng
•	Quản lý đơn nhập
•	Theo dõi thống kê
